# dash_vendas_excel
Projeto de criação de um Dash de Vendas no Excel, com foco na organização, análise e visualização estratégica de dados. A proposta foi transformar dados brutos em informações claras e intuitivas, por meio de indicadores, gráficos dinâmicos e filtros interativos, permitindo uma análise eficiente do desempenho comercial e apoio à tomada de decisão.
